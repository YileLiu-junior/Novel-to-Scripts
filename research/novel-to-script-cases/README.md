# Novel to Script Case Materials

This folder contains five research cases selected for the AI novel-to-script product benchmark.

Scoring dimensions used in the Feishu sheet:

- Data completeness
- Adaptation comparison value
- Input slicing friendliness
- Narrative complexity
- Script format usability

Copyright/commercial-risk scoring was intentionally removed per the research-only requirement.

## Cases

1. `01_sherlock_study_in_pink`
   - Novel: `a-study-in-scarlet_novel.txt`
   - Script: `sherlock-a-study-in-pink_script.pdf`
   - Use: detective novel modernization, one-hour TV structure, clue compression.

2. `02_little_women_2019`
   - Novel: `little-women_novel.txt`
   - Script: `little-women-2019_script.pdf`
   - Use: long novel compression, ensemble cast, nonlinear restructuring.

3. `04_pinocchio_2022`
   - Novel: `pinocchio_novel.txt`
   - Script: `pinocchio-2022_script.pdf`
   - Use: fairy tale reinterpretation, fantasy/action/music script treatment.

4. `03_pride_and_prejudice_2005`
   - Novel: `pride-and-prejudice_novel.txt`
   - Script: `pride-and-prejudice-2005_script.pdf`
   - Use: dialogue, subtext, social-scene dramatization.

5. `05_wizard_of_oz_1939`
   - Novel: `wizard-of-oz_novel.txt`
   - Script: `wizard-of-oz-1939_script.pdf`
   - Use: journey structure, spectacle scenes, musical/fantasy adaptation.
