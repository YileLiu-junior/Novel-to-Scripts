# Web Series

**原著**: Ch 1、Ch 2、Ch 3

## 1. 贝克街房间 内景 日

华生打量房间，福尔摩斯已经把注意力放在更细微的观察上。

**约翰·华生**  
*（他把手杖靠到椅边，目光仍停在福尔摩斯身上）*  
如果房租真的能分摊，我愿意试试。

**歇洛克·福尔摩斯**  
*（他没有坐下，只是扫了一眼华生的手杖）*  
那我们先弄清彼此不能忍受什么。


## 2. 贝克街客厅 内景 晨

一封来自苏格兰场的信被递进来，福尔摩斯的倦怠立刻被案件点亮。

**歇洛克·福尔摩斯**  
*（他迅速套上外衣，语速比刚才快了一拍）*  
终于有件事不像例行公事。
