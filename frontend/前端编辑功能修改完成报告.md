## 2. 编辑功能说明

### 人物管理
- **可编辑字段**：name、aliases（逗号分隔输入，保存转列表）、narrative_role、voice_profile.rhythm、voice_profile.diction
- **编辑交互**：点击"编辑"→ 显示表单（ID 只读）→ 保存/取消
- **保存后**：更新本地 project，清空编辑状态，页面刷新

### 场景管理
- **可编辑字段**：title、scene_heading.text、location.name、location.time、interior_exterior（selectbox）、dramatic_purpose（多行文本，保存转列表）、action（多行文本，保存转列表）
- **编辑交互**：点击"编辑"→ 显示表单（ID 只读）→ 保存/取消
- **保存后**：更新本地 project，清空编辑状态，页面刷新

### 情节页面
- **可编辑字段**：title、description（同步更新 summary）、type、importance
- **只读展示**：characters、source_refs（标注 TODO 后续完善）
- **编辑交互**：点击"编辑"→ 显示表单（ID 只读）→ 保存/取消
- **保存后**：更新本地 project，清空编辑状态，页面刷新

## 3. ID 与名称处理

- **ID 不可编辑**：编辑表单中 ID 仅作为只读标题展示（如"编辑人物（ID: char_001）"），不提供输入框
- **内部引用仍用 ID**：所有数据关联（关系边、场景规划、因果关系）内部继续使用 ID
- **关系展示用名称**：人物关系、场景规划、因果关系在展示时，通过实时构建的 `{id: name}` 映射将 ID 转换为名称，找不到名称时回退显示原 ID

## 4. 重名校验

| 页面 | 校验字段 | 校验规则 |
|---|---|---|
| 人物管理 | name | 非空 + 不与其他人物重复（排除自身，大小写不敏感） |
| 场景管理 | title | 非空 + 不与其他场景重复（排除自身，大小写不敏感） |
| 情节页面 | title | 非空 + 不与其他情节重复（排除自身，大小写不敏感） |

重复时显示中文错误提示："人物名称不能重复" / "场景名称不能重复" / "情节名称不能重复"。

## 5. 前端状态更新

- **编辑保存**：只调用 `storage.update_project()` 更新本地 JSON，不调用后端接口
- **删除**：二次确认后从列表移除，调用 `storage.update_project()` 保存，不调用后端接口
- **刷新页面**：编辑内容不会保留（恢复后端原始数据或本地 JSON 中的数据）
- **关系同步**：名称映射基于当前前端状态实时生成，编辑名称后关系展示自动更新
